Korte uitleg: Wat zijn HTML, CSS en JavaScript?

HTML - De structuur van je webpagina.

HTML staat voor HyperText Markup Language. Het is de basis van elke website. Met HTML bepaal je wat er op je pagina staat: teksten, afbeeldingen, knoppen, enzovoort.

CSS ‚Äì De stijl en opmaak.

CSS staat voor Cascading Style Sheets. Met CSS geef je je HTML een opmaak en vorm: kleuren, lettertypes, lay-out, enzovoort.

JavaScript ‚Äì De logica en interactie.

JavaScript zorgt ervoor dat je interactie kunt toevoegen aan je website en dat je pagina‚Äôs tot leven komen. Je gebruikt het ook om achterliggende logica uit te voeren, zoals het bijhouden van de score in je game.
