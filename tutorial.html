Workshop: Maak je eigen memory game in HTML, CSS en JavaScript

  • Duur: 2 uur
  • Niveau: Beginner – geen voorkennis nodig

Introductie

Welkom bij deze workshop! Vandaag gaan we samen stap voor stap een twee-speler memory game bouwen met behulp van HTML, CSS en JavaScript – de drie belangrijkste technologieën van het web. Deze workshop is speciaal bedoeld voor beginners. Je hoeft dus nog niets te weten over programmeren!
